What is Join?

Join is a dynamic Kanban board. In japanese "Kanban"means "Visual Signal" or "Visual Card". The main purpose of Kanban is to help and improve manufacturing efficiency and create a transparent work process for everyone involved.


How to use it

Board Board panel usage
1. To see all the tasks in process go to . Here you find all created task and their current status.
2. You can change the status of a task by drag and drop. Grab the task to change and drop it in the collum of the new status.
3. For more details of a task or to edit it, click on the task. In the opened Overlay you can see the whole information of the task. To edit the task press the button. Change the fields to edit and press the button. Here you can also delete the task by pressing the button and follow the instructions.
4. If you want to create a new task with a preselected status, press the at the status you want to add the task. Then just follow the instructions of the "Add task" panel.
5. To search or filter the tasks, you can use the search bar. not implemented now
Add Task Add task panel usage
1. To create a new task go to , fill in the required fields and press the button. The new task is pushed into the "To do" status on board by default.
Contacts Contacts panel usage
1. To create a new contact go to and press the button. Fill in the required fields and press the button.
2. To edit a contact go to and select the contact to edit. Press the button. Change the fields to edit and press the button. Here you can also delete the selected contact by pressing the button and follow the instructions.
